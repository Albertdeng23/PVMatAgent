## 🎯 二、推荐材料：CsSn₀.₄Ge₀.₆I₃

### 2.1 成分设计逻辑

基于文献报道的 HSE06 DFT 精确数据，CsSn₁₋ₓGeₓI₃ 体系的带隙随 Ge 含量线性递增：

```
x = 0.00 (CsSnI₃)          → 1.30 eV
x = 0.25 (CsSn₀.₇₅Ge₀.₂₅I₃) → 1.48 eV
x = 0.50 (CsSn₀.₅Ge₀.₅I₃)   → 1.69 eV  ← 接近但略低
x = 0.60 (CsSn₀.₄Ge₀.₆I₃)   → ~1.75 eV ← 🎯 目标！
x = 0.75 (CsSn₀.₂₅Ge₀.₇₅I₃) → 1.84 eV
x = 1.00 (CsGeI₃)          → 1.93 eV
```

**线性插值验证：1.69 + (1.84−1.69) × (0.60−0.50)/(0.75−0.50) = 1.75 eV ✅**

## ⚠️ 五、风险与挑战

### 🔴 主要风险

1. **结构稳定性临界**：容差因子 t ≈ 1.055（理想 0.9–1.0），八面体因子 μ ≈ 0.624（阈值 0.645）。Ge含量60%时，结构偏向六方/低维相的倾向增强。

2. **Sn²⁺ 氧化**：这是所有锡基钙钛矿的"阿喀琉斯之踵"。Sn²⁺ → Sn⁴⁺ 氧化引入深能级陷阱，加速器件退化。

3. **Ge 成本高昂**：锗是稀有元素，规模化应用受限。

### 🟢 缓解策略（文献验证）

| 策略 | 机制 |
|------|------|
| Ge²⁺ 共掺杂 | Ge的天然氧化层钝化表面，抑制体相Sn氧化（CsSn₀.₅Ge₀.₅I₃稳定性优于纯CsSnI₃） |
| SnF₂ 添加剂 | 填充阳离子空位，降低背景空穴浓度 |
| 封装技术 | 隔绝氧气和水分 |
| 抗氧化添加剂 | 如 SnCl₂、肼类还原剂 |
